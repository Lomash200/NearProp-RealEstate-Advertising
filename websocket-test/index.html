Connection Troubleshooting

Common Issues

  • Server not running: Make sure the Spring Boot application is running on port 8080
  • Invalid JWT token: Your token might be expired or invalid
  • CORS issues: The server might be blocking cross-origin requests
  • WebSocket 1006 error: This typically means abnormal closure - the server is rejecting the connection
  • 403 Forbidden errors: Your token might not have the required permissions

NearProp-Specific Tips

  • Correct URL format: According to the server configuration, use ws://localhost:8080/api/ws?token=your_jwt_token
  • STOMP topics: Subscribe to /topic/chat/{roomId}, /topic/chat/{roomId}/typing, and /user/queue/messages
  • Send destinations: Use /app/chat/{roomId}/send for regular messages or /app/admin/chat/{roomId}/send for admin messages
  • Message format: {"content": "Your message text"}
  • JWT token format: The token should be sent as a URL parameter, not in headers
